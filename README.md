# Poetry Printer (詩の印刷)

## 概要・説明
Poetry Printer（詩の印刷）は、詩や文章を美しくレイアウトし、そのままA4用紙に印刷・PDF保存ができるブラウザベースのテキスト組版アプリケーションです。

エレガントで落ち着いた配色のUI（ユーザーインターフェース）を備え、入力したテキストをリアルタイムでプレビューできます。縦書き・横書きの切り替えをはじめ、豊富な日本語フォントの選択、文字サイズ、行間、字間などの細やかなタイポグラフィ調整が可能です。フレームワークを使用しない軽量なシングルページアプリケーション（SPA）として構築されています。

## 主な機能一覧
- **リアルタイムプレビュー**: 入力したテキストや設定の変更が即座に右側のA4キャンバスに反映されます。
- **柔軟なテキスト設定**: タイトル、作者名、詩の本文をそれぞれ分けて入力可能。
- **縦書き・横書き対応**: 日本語の美しい縦組み（右から左）と横組みをワンクリックで切り替え。
- **豊富なフォント選択**: しっぽり明朝、Noto Serif JP、Zen Old Minchoなど、詩の雰囲気に合わせたフォントを選択可能。
- **詳細なタイポグラフィ調整**: 
  - 本文とタイトルの文字サイズ調整
  - 行間、字間（トラッキング）の微調整
  - 余白（上下位置）の調整
  - 文字揃え（左寄せ、中央揃え、右寄せ）
- **デザインのカスタマイズ**: 本文や作者名の文字色をカラーピッカーで自由に変更。タイトル下の装飾線の表示・非表示の切り替え。
- **印刷最適化（Print CSS）**: 「印刷する」ボタンを押すとサイドバーが非表示になり、A4サイズにぴったり収まるよう自動的に最適化されて印刷（またはPDF出力）ダイアログが起動します。

## 使用技術・ライブラリ
当プロジェクトは外部のJavaScriptフレームワーク（ReactやVueなど）に依存せず、ピュアなウェブ標準技術のみで構築されています。

- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript
- **Webフォント**: Google Fonts API
  - 和文: Shippori Mincho, Noto Serif JP, Zen Old Mincho, Noto Sans JP
  - 欧文（UI用）: Cormorant Garamond

## セットアップ・使い方

### セットアップ
外部ライブラリのインストールやビルド環境の構築は一切不要です。

1. 本ソースコードを `index.html` という名前で保存します。
2. 保存した `index.html` をお好みのモダンブラウザ（Google Chrome, Safari, Edge, Firefoxなど）で開きます。

### 使い方
1. **テキストの入力**: 画面左側のサイドバーにて、「タイトル」「作者名」「詩の本文」を入力します（タイトルと作者名は省略可能です）。
2. **スタイルの調整**: 
   - 「フォント」や「文字の向き（縦書き/横書き）」「文字揃え」を選択します。
   - 各種スライダーを動かして、文字サイズ、行間、字間、上下の余白を直感的に調整します。
   - 必要に応じて文字の色を変更したり、装飾線のオン/オフを切り替えます。
3. **印刷・PDF保存**: 右側のプレビューで理想のレイアウトが完成したら、サイドバー下部の「印刷する」ボタンをクリックします。ブラウザの印刷ダイアログが開くので、そのままプリンターで印刷するか、「PDFとして保存」を選択してデータとして書き出してください。
